# HoursReport

Repository for the third challenge of the Elixir's path from [Rocketseat Ignite](https://rocketseat.com.br).

## :rocket: Techs

* [Elixir](https://elixir-lang.org/)

## :wrench: Setup

  * Install dependencies with `mix deps.get`
  * For interactive testing use `iex -S mix`

## :white_check_mark: Test

To run tests:

* Run with `mix test --cover`
